## 📖 项目简介

智学导师是一个面向中小学生和教育工作者的智能学习辅助系统，通过结合DeepSeek大语言模型的强大能力，为学生提供个性化的学习支持、智能题目解答和作业批改等功能。本系统旨在提高学习效率，增强学习体验，同时收集和分析学习数据，为教育工作者提供数据支持。

## ✨ 主要功能

### 🧩 智能题目解答
- **文本题目解答**：输入任何学科的文本题目，获取详细解答和解题思路
- **图片题目解答**：上传题目图片，智能识别并解答，支持手写题目
- **问题历史记录**：保存所有解答过的题目，方便复习和查看

### 📚 智能作业批改
- **作业提交与批改**：上传作业图片，获取AI批改结果和评分
- **详细反馈与建议**：针对作业中的问题提供具体反馈和改进建议
- **作业批改历史**：记录所有批改过的作业，跟踪学习进度

### 💡 知识卡片系统
- **创建知识卡片**：记录重要知识点和学习笔记
- **从题目生成**：自动从已解答的题目中提取知识点，生成知识卡片

### 🔍 学习资源库
- **多样化学习资源**：提供视频、文章、图书、练习题等多种形式的学习资源
- **个性化推荐**：基于用户学习历史和偏好，智能推荐相关学习资源
- **资源收藏与评价**：用户可以收藏和评价学习资源，帮助其他用户选择

### 📊 学习分析
- **学习数据分析**：统计和分析用户的学习数据，生成可视化报告
- **学习风格识别**：识别用户的学习风格和特点，提供个性化学习建议
- **学科能力评估**：评估用户在各个学科领域的能力水平

### 📋 学习历史记录
- **综合学习记录**：记录所有学习活动，包括题目解答、作业批改、知识卡片等
- **学习进度追踪**：追踪用户的学习进度和成长轨迹
- **数据导出和共享**：支持学习数据导出和与教师共享

## 🛠️ 技术栈

- **后端框架**：Django
- **前端框架**：Bootstrap 5
- **数据库**：SQLite (开发) / PostgreSQL (生产)
- **AI模型**：DeepSeek大语言模型
- **数据可视化**：ECharts
- **版本控制**：Git

## 🚀 安装和使用

### 系统要求
- Python 3.8+
- pip
- virtualenv (建议使用)

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/smart-learning-assistant.git
cd smart-learning-assistant
```

2. 创建并激活虚拟环境
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 初始化数据库
```bash
python manage.py migrate
```

5. 创建超级用户
```bash
python manage.py createsuperuser
```

6. 添加示例数据（可选）
```bash
python manage.py add_sample_resources
```

7. 启动开发服务器
```bash
python manage.py runserver
```

8. 访问系统
在浏览器中访问 http://127.0.0.1:8000/

## 功能筛选说明

系统提供了多种筛选功能来帮助用户快速找到所需内容：

1. **问题筛选**：
   - 按问题类型（文本/图片）筛选

   - 按难度筛选
   - 关键词搜索

## 系统特色

1. **智能化**：利用先进AI技术实现智能解题和作业批改
2. **个性化**：根据用户学习情况提供个性化的知识卡片和资源推荐
3. **系统化**：将零散的知识点系统化整理，形成完整的知识体系
4. **可视化**：通过图表和统计数据直观展示学习进度和成果
5. **互动性**：用户可以评价和分享学习资源，促进交流与合作
6. **视觉统一**：整个系统采用一致的蓝色主题设计（#4e6ef2），所有页面元素（包括知识卡片、问题列表、作业批改结果等）都保持统一的视觉风格，提供舒适的用户体验

## 用户界面设计

- **配色方案**：系统采用蓝色（#4e6ef2）为主色调，搭配白色背景和灰色辅助色，营造专业、简洁的学习环境
- **卡片设计**：所有卡片元素（知识卡片、问题卡片、作业卡片）采用统一的圆角矩形设计，配以轻微阴影效果，提升页面层次感
- **响应式布局**：系统界面自适应不同屏幕尺寸，在电脑、平板和手机上均可流畅使用
- **无障碍设计**：考虑色盲和弱视用户的需求，确保系统在不同视觉条件下仍可正常使用

## 常见问题

1. **为什么筛选功能不生效？**
   - 请尝试清除浏览器缓存或强制刷新页面(Ctrl+F5)
   - 确保选择了正确的筛选选项后点击了筛选按钮

2. **为什么知识卡片内容显示不正确？**
   - 系统使用Markdown格式显示内容，请确保内容格式正确
   - 如遇问题，可尝试编辑知识卡片重新保存

3. **系统支持哪些题目类型？**
   - 文字题目：直接输入题目文本
   - 图片题目：上传题目图片（支持jpg, png, jpeg格式）

4. **为什么知识卡片颜色显示异常？**
   - 请确保您的浏览器已更新到最新版本
   - 尝试清除浏览器缓存或使用无痕/隐私模式访问
   - 如果使用了自定义浏览器主题或暗黑模式，可能会影响显示效果
