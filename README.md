## 项目简介

智学导师是一个基于人工智能的教育辅助系统，旨在为中小学生提供智能化的学习支持服务。系统集成了题目解答、作业批改、知识卡片生成等功能，帮助学生更高效地学习和掌握知识点，同时为教师提供智能化的教学辅助工具。

## 核心功能

### 📝 题目解答
- 支持文字题目和图片题目的智能解答
- 提供详细的解题思路和答案
- 记录解题历史，方便复习和回顾

### 📚 作业批改
- 智能识别和批改各类作业题目
- 提供评分和详细的改进建议
- 多选题自动识别和分析

### 💡 知识点提取
- 从题目和作业中提取关键知识点
- 生成知识卡片，便于记忆和理解
- 智能推荐相关学习资源

### 📊 学习数据分析
- 统计学习成果和进度
- 分析学习强项和弱项
- 个性化学习建议

## 技术特点

- **智能OCR**: 准确识别手写文字和印刷体
- **AI分析引擎**: 智能分析题目类型和解题方法
- **个性化推荐**: 根据学习历史推荐相关资源
- **响应式设计**: 适配各种设备屏幕尺寸
- **用户友好界面**: 简洁直观的操作体验

## 快速开始

### 环境要求
- Python 3.8+
- Django 4.0+
- PostgreSQL 13+

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/smart-tutor.git
cd smart-tutor
```

2. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 配置数据库
```bash
python manage.py migrate
```

5. 创建管理员账户
```bash
python manage.py createsuperuser
```

6. 启动服务器
```bash
python manage.py runserver
```

访问 http://127.0.0.1:8000/ 开始使用智学导师系统。

## 系统架构

智学导师系统采用Django框架开发，主要由以下模块组成：

- **用户管理**: 用户注册、登录、个人资料管理
- **题目解答**: 文字题目和图片题目的解答功能
- **作业批改**: 智能识别和批改作业
- **学习记录**: 题目和作业的历史记录
- **知识库**: 知识点提取和知识卡片生成
- **资源推荐**: 个性化学习资源推荐

## 用户指南

### 新用户注册
1. 点击首页"注册"按钮
2. 填写基本信息（用户名、邮箱、密码）
3. 验证邮箱
4. 完善个人资料（学校、年级、感兴趣科目）

### 解答题目
1. 点击"解答题目"按钮
2. 选择题目类型（文字题或图片题）
3. 输入题目内容或上传题目图片
4. 提交后等待系统分析和解答

### 批改作业
1. 点击"批改作业"按钮
2. 上传作业图片
3. 等待系统自动识别和批改
4. 查看批改结果和建议

### 查看学习历史
1. 通过主页访问"学习历史"
2. 选择"题目历史"或"作业历史"
3. 浏览历史记录，点击查看详情

## 功能筛选说明

1. **问题筛选**：
   - 按问题类型（文本/图片）筛选

3. **知识卡片筛选**：

   - 关键词搜索

4. **学习资源筛选**：
   - 按资源类型筛选
   - 按难度筛选
   - 关键词搜索

## 系统特色

1. **智能化**：利用先进AI技术实现智能解题和作业批改
2. **个性化**：根据用户学习情况提供个性化的知识卡片和资源推荐
3. **系统化**：将零散的知识点系统化整理，形成完整的知识体系
4. **可视化**：通过图表和统计数据直观展示学习进度和成果
5. **互动性**：用户可以评价和分享学习资源，促进交流与合作
6. **视觉统一**：整个系统采用一致的蓝色主题设计（#4e6ef2），所有页面元素（包括知识卡片、问题列表、作业批改结果等）都保持统一的视觉风格，提供舒适的用户体验

## 用户界面设计

- **配色方案**：系统采用蓝色（#4e6ef2）为主色调，搭配白色背景和灰色辅助色，营造专业、简洁的学习环境
- **卡片设计**：所有卡片元素（知识卡片、问题卡片、作业卡片）采用统一的圆角矩形设计，配以轻微阴影效果，提升页面层次感
- **响应式布局**：系统界面自适应不同屏幕尺寸，在电脑、平板和手机上均可流畅使用
- **无障碍设计**：考虑色盲和弱视用户的需求，确保系统在不同视觉条件下仍可正常使用

## 常见问题

1. **为什么筛选功能不生效？**
   - 请尝试清除浏览器缓存或强制刷新页面(Ctrl+F5)
   - 确保选择了正确的筛选选项后点击了筛选按钮

2. **为什么知识卡片内容显示不正确？**
   - 系统使用Markdown格式显示内容，请确保内容格式正确
   - 如遇问题，可尝试编辑知识卡片重新保存

3. **系统支持哪些题目类型？**
   - 文字题目：直接输入题目文本
   - 图片题目：上传题目图片（支持jpg, png, jpeg格式）

4. **为什么知识卡片颜色显示异常？**
   - 请确保您的浏览器已更新到最新版本
   - 尝试清除浏览器缓存或使用无痕/隐私模式访问
   - 如果使用了自定义浏览器主题或暗黑模式，可能会影响显示效果
